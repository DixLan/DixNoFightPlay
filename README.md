## Description

------

This script was made for AragonRP, a Frenche militaryRP server based on FiveM. They spawn ped on different location for PvE fighting.



## Requirements

------

- **ESX:Notifiacation**

  

## Download & Install

------

- Download and extract the package

* Drop the `DixNoFightPlay-master` directory into you `resources` directory
* Rename the directory from `DixNoFightPlay-master` to `Dsupershop`
* Add `ensure DixNoFightPlay` in your `server.cfg`
* Edit `config.lua` to your liking
* Start your server and enjoy !
